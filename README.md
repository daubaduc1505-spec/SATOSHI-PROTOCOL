## ✨ What is This?

A **fully automated reward distribution system** built on Solana that:

- 💰 **Claims fees** from Pump.fun trades automatically
- 🔄 **Swaps SOL → BTC** using Jupiter DEX
- 🎁 **Distributes BTC** to token holders every 60 minutes
- 🤖 **Telegram bot** sends real-time notifications
- 🎨 **Pixel-style dashboard** with falling BTC animations

---

## 🔥 Key Features

| Feature | Description |
|---------|-------------|
| **🎯 80/20 Model** | 80% of fees go to holders, 20% to development |
| **⏱️ 60-Minute Cycles** | Automatic distribution every hour |
| **💰 Percentage-Based** | More tokens = more rewards |
| **🔐 2-Wallet System** | Separate wallets for fees and rewards |
| **📱 Telegram Bot** | Check stats, leaderboard, eligibility |
| **🎵 EDM Music** | 24/7 background music (toggle on/off) |
| **₿ Falling BTC** | Pixel animation with floating Bitcoin |
| **📊 Live Stats** | Real-time dashboard with USD conversion |
